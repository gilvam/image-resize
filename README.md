# image-resize

Redimensiona uma imagem para um tamanho aproximado do escolhido. 

```javascript
const max = {
 	image: 3.6 * 1000000, // 3.6mb tamanho máximo bruto
 	big: 1920, // altura ou largura máxima ( maior valor )
 	small: 1080, // altura ou largura máxima ( menor valor )
 };
```
